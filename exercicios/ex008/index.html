Principais formatações

Negrito

Aqui podemos ver uma palavra em negrito usando a tag (p), não semântica

Aqui podemos ver uma palavra em negrito usando a tag (strong), tag semântica

Parece não ter diferença mas no html5 a semântica faz toda a diferença ja que o próprio html5 foi atualizado para isso. Neste caso a tag STRONG significa FORTE

Itálico

Aqui podemos ver uma palavra em itálico usando a tag (t), não semântica

Aqui podemos ver uma palavra em itálico usando a tag (em), tag semântica

Embora pareçam iguais, são escritar de formas diferentes justamente para dar significado, por isso se chama HTML SEMÂNTICO. Neste caso a tag EM significa ÊNFASE
